Oracle-回滚已提交的数据

Oct 25, 2018 | 技巧篇
文章目录

  • 第一步：查询所需表的最后需改时间

    select created, last_ddl_time from user_objects where object_name = ‘表名’

    其中 created 为表的创建时间，last_ddl_time 为最后修改时间

  • 第二步：回滚数据到最后操作时间之前

    flashback table 表名 to timestamp to_timestamp(‘2018-08-09 17:06:00’, ‘yyyy-mm-dd hh24:mi:ss’)；

  • 如果提示：“因为未启用行移动功能，不能闪回表”，则执行下面的语句后再执行第二步操作：

    alter table 表名 enable row movement;
